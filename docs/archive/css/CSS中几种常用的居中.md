## 居中

### 行内元素水平居中

```
text-align:center;
```

### 文字相对图片、输入框垂直居中

```
vertical-align:middle
```

### 块元素水平居中

```
margin:0 auto;
```

### 单行文字垂直居中

```
height:n px;
line-height:n px;
```

更灵活的布局方式当然是建议你使用 flex 布局，[查看 Flex 布局](https://www.jianshu.com/p/7cfb990d8dfd)
