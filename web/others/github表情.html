# github 表情

# 在 commit 时添加一个 emoji 表情图标

git commit -m ':emoji: 此次提交的内容说明'
1

# 添加多个 emoji 表情图标

git commit -m ':emoji1: :emoji2: :emoji3: 此次提交的内容说明'
1

在提交内容的前面增加了 emoji 标签： :emoji:，其中 emoji 是表情图标的标签，列表见下面的附录表格。
